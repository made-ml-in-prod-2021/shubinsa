## Production-ready ML project

* Installation: ```python -m venv .venv .venv\Scripts\activate.bat pip install -r requirements.txt```

LogisticRegression:
* Train:
```python ml_project/train_pipeline.py configs/train_config_lr.yml```
* Predict:
```python ml_project/predict_pipeline.py configs/train_config_lr.yml```

RandomForest:
* Train:
```python ml_project/train_pipeline.py configs/train_config_rf.yml```
* Predict:
```python ml_project/predict_pipeline.py configs/train_config_rf.yml```

Tests:
```pytest tests```
